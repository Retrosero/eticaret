# @eticart/ui

Ortak React bileşenleri.

Şu anda tek bir `TrCurrency` bileşeni (Türk Lirası para biçimleyici) içerir.
Faz 5'te vitrin/tema bileşenleri eklenir.

Tüm bileşenler:
- WCAG 2.1 AA uyumlu
- Türkçe erişilebilirlik etiketleri
- `forwardRef` destekli
- Sunucu bileşeni uyumlu (Next.js App Router)
